## 初步理解

当前任务是研究型任务，不是直接实现业务代码。核心问题是：参考多个开源记忆系统（Open-source Memory Systems），判断现有桌宠原生记忆内核（companion-native Memory Kernel）应该如何优化，尤其关注事件记忆（Episodic Memory）、语义记忆（Semantic Memory）、关系记忆（Relationship Memory）、记忆图谱（Memory Graph）和反思记忆（Reflective Memory）。
