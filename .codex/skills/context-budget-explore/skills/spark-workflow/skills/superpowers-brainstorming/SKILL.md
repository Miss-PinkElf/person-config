# Superpowers Brainstorming（内置版）

Align 的意义，是避免“很快地做错”。

## 必须行为

1. 先扫描相关代码、文档和已有模式
2. 一次只问一个问题
3. 可多选时优先给选项
4. 给出 2-3 个方案、取舍与推荐
5. 把已选方向整理成可进入 proposal 的摘要
6. 在继续之前等待用户明确确认

读取 [../../references/brainstorming-guide.md](../../references/brainstorming-guide.md)。

## 输出

- 已确认约束
- 推荐方案
- 剩余开放问题
- 可直接进入 proposal 的方向摘要

## 约束

- 不要因为任务看起来小就跳过 alignment
- 不要直接跳进实现
- 如果对话仍是纯探索、且还不需要承诺方向，转去 [../openspec-explore/SKILL.md](../openspec-explore/SKILL.md)
