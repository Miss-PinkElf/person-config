## 后端需求

<font style="color:rgb(31, 35, 40);">桌面陪伴机器人，可以偶尔跑到我自己的鼠标上，也可以直接快捷键打开对话框。形象可以是一个live2d，有rag形成记忆，mcp作为手脚，llm作为大脑</font>

运行环境： win / mac（暂定优先win）

（总的来说有点像 社会学分析）（需要有开关，那些隐私不能记录，像一个插件一样，可以插拔）

（目前还没有很好的解决方案，对于记忆）

### 后端需要做的

- 获取常见浏览器的记录 / 标签页的访问时长 / nfsw去除 / 记录需要访问什么 / 喜欢在什么时间点访问什么，
  - 目的：了解人的喜好，给这个人画像
- 获取电脑，整体的应用的使用时间，什么时候使用，使用的时间段是哪些
  - 不用实时获取，也不需要实时截图，每一天的结束自动记录
- 环境麦克风的突然增加，情绪的突然激动
  - 获取当前屏幕的内容，结合摄像头的表情，环境音，音乐，进行分析
    - 然后做出回应
- 记录常见的习惯，比如喜欢在饭点看什么视频，喜欢在深夜敲代码，类似的
- 可以自动化帮你做一些事情，这个自动化不用用户同意，做之前判断一下，用户正在干什么，当前的的时间是？当前的行为是？ 然后再执行，需要判断自动执行后的反应，根据这个反应来继续，如果用户不满意，表现出不满，可能进行道歉，想一个人一样，满意的话，可以寻求夸奖。

总之我想让这个桌宠，像一个人，而不是一个工具，可以不断的了解，你，而不是像一个程序一样，事事都需要向你确认，来陪伴你，可能不能敲代码。可能不能帮你解决问题，但是它一定是最了解你的，一只默默在电脑上陪伴你，和你打趣
